# 質問ログ

学習中に出てきた「わからない」を Claude Code に聞いたときの記録。
**要点だけ** 残す（質問の趣旨 + 回答の要点 3〜4 行）。長文回答は割愛。

同じ関連章で **「補強済み」になっていないエントリが 3 件たまる** と、Claude が回答の末尾で「補強ページ作りますか？」と提案します。承認したら `extras/supplements/` に番外編が作られ、ここの該当エントリの「補強済み」が `☑` になります。

このログは、後で読み返したときに「あれ、これ前にも聞いたな」と気づくための備忘でもあります。

## 記録フォーマット

```
### YYYY-MM-DD: <質問の見出し>
- 関連章: ChN（または「全体」）
- 質問: <ユーザーが投げた質問の趣旨>
- 回答要点:
  - <要点 1>
  - <要点 2>
  - <要点 3>
- 補強済み: ☐
```
