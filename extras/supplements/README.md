# 補強ページ（番外編）

`extras/qa-log.md` に **同じ関連章の質問が 3 件以上たまったとき**、Claude Code がここに補強ページを作ります。本編（`chapters/`）の章をいじるのではなく、別ファイルとして「○○をもう一歩」のような形で残すための置き場です。

## ページのフォーマット

各補強ページ（`NNN-タイトル.md`）は下記の構成。

```
# 番外編 NNN: タイトル

## きっかけ
- qa-log にあった 3 件の質問の見出しを列挙

## 解説
本編と同じく、新規の専門用語には身近な実例を添える。

## 重要な単語まとめ（あれば）

## 関連する本編へのリンク
```
